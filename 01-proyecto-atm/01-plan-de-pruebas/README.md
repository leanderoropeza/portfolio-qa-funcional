# Plan de Pruebas – ATM

Esta carpeta contiene el **plan de pruebas completo** del proyecto ATM.

## Contenido

- `Plan-de-Pruebas.md`: Documento con alcance, objetivos, ambiente, precondiciones, riesgos, criterios de entrada/salida, matriz de priorización y calendario de ejecución.

## Recomendaciones

- Revisar el plan antes de ejecutar cualquier caso de prueba.
- Usar las tablas de priorización y métricas para seguimiento de ejecución.
- Referenciar los casos de prueba usando los IDs definidos en `02-casos-de-prueba/`.
