# Automatización – ATM

Esta carpeta contiene ejemplos de **automatización de pruebas**.

## Contenido

- `java-selenium-example/`: Proyecto Maven con ejemplos de pruebas UI (ej.: `WithdrawMoneyTest.java`).
- `postman/`: Colección Postman para pruebas API del backend (`atm_api_collection.json`).

## Uso

1. Para Selenium: abrir proyecto Maven en IDE, ejecutar tests locales.
2. Para Postman: importar colección y ejecutar pruebas API.
3. Instrucciones de CI disponibles en el README dentro de cada subcarpeta.
