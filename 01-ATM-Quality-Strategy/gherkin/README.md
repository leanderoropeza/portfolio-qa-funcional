# Gherkin – ATM

Esta carpeta contiene los **escenarios BDD** en formato Gherkin para flujos críticos del ATM.

## Contenido

- Archivos `.feature` (ej.: `withdraw_money.feature`, `atm_error_handling.feature`).

## Uso

- Usar como base para pruebas automatizadas.
- Facilita trazabilidad entre historias, casos y ejecución automatizada.
