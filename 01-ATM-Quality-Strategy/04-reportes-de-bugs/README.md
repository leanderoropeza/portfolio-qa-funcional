### 📝 Estructura de mis Reportes
Cada reporte individual sigue este estándar para asegurar una resolución rápida:
1.  **Pasos para Reproducir:** Secuencia exacta para replicar el fallo.
2.  **Resultado Esperado vs. Resultado Real:** Contraste claro del error.
3.  **Ambiente de Prueba:** Versión del software y datos de prueba utilizados.
4.  **Evidencia Visual:** Capturas de pantalla o logs adjuntos.
