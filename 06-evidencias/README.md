# Evidencias de QA

Esta carpeta contiene evidencias que respaldan la ejecución de pruebas y la detección de bugs durante el proyecto de demostración.

Tipos de evidencias:

- **Capturas de pantalla**: muestran errores y validaciones de la aplicación.
- **Logs**: registros de ejecución que permiten analizar comportamientos inesperados.
- **Videos**: grabaciones de flujos completos o bugs críticos.
- **PDFs**: reportes exportados de herramientas de seguimiento de incidencias.

Estas evidencias permiten demostrar la trazabilidad, el seguimiento y la calidad de las pruebas realizadas.
