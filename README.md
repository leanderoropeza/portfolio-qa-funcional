# 🚀 Framework de Automatización E2E | E-commerce Project
> **Arquitectura de pruebas escalable diseñada para la validación de flujos críticos.**

Este proyecto demuestra la implementación de un framework de automatización robusto. No es solo un script; es una estructura profesional preparada para integrarse en pipelines de **CI/CD**.

---

## 🛠️ Stack Tecnológico
*   **Lenguaje:** Java (JDK 11+)
*   **Automatización:** Selenium WebDriver + TestNG
*   **Patrón de Diseño:** **Page Object Model (POM)**
*   **Reportes:** ExtentReports para visibilidad de resultados.

---

## 🏗️ Alcance Técnico
*   **Manejo de Esperas:** Implementación de Explicit Waits para evitar tests "flaky".
*   **Modularidad:** Separación de localizadores y lógica de negocio.
*   **Escenarios:** Login, gestión de carrito y flujo de checkout completo.

> **Nota:** Este proyecto valida mi capacidad para automatizar cualquier plataforma web bajo estándares de industria.
