# Reportes de Defectos – ATM

Esta carpeta contiene todos los defectos detectados durante la ejecución de pruebas.

## Contenido

- Archivos `.md` por cada defecto (ej.: `DF-ATM-001.md`).
- Cada reporte incluye: título, severidad, estado, pasos para reproducir, resultado esperado, resultado real y evidencia.

## Uso

- Consultar los defectos para seguimiento y cierre.
- Relacionar con casos de prueba y matriz de trazabilidad.
